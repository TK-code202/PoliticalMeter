# _Political Meter_

#### _This website checks what political side users tilt towards._

#### By _**Tofunmi Kusimo**_

## Technologies Used

* _GIT_
* _HTML5_
* _CSS_
* _BOOTSTRAP 4.5.3_
* _Javascript and jQuery_


## Description

_This website checks which side of the typical American political spectrum, users are on based on their beliefs which is gathered through information they provide to certain questions on the website. The website determines their political side using branching._

## Setup/Installation Requirements

* _Clone this repository to your desktop_
* _Navigate to the top level of the directory_
* _Open the index.html file in your browser._
